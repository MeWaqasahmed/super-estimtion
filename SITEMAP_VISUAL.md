## 🔄 User Journey Maps

### Journey 1: New Visitor → Lead
```
1. Land on Homepage (Google Search)
2. Read "Why Choose Us" section
3. Click "Services" in navigation
4. Review relevant service
5. Click "Get Quote" CTA
6. Fill contact form
7. Submit → LEAD CAPTURED ✅
```

### Journey 2: Returning Visitor → Customer
```
1. Return to website (bookmark)
2. Navigate to "Pricing"
3. Review pricing tiers
4. Check "FAQ" for questions
5. Read testimonials for trust
6. Call phone number → CUSTOMER ✅
```

### Journey 3: Research → Educated Lead
```
1. Find blog article (Google)
2. Read full article
3. Click related article
4. Navigate to "About Us"
5. Review credentials
6. Submit contact form → LEAD CAPTURED ✅
```
